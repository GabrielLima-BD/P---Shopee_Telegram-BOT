📌 O QUE FOI MUDADO:

✅ Tokens agora são HARD-CODED (embutidos no código)
✅ Não precisa mais de .env
✅ Não precisa mais de pastas junto
✅ .exe funciona sozinho em qualquer lugar
✅ Cria pastas automaticamente onde for executado


📌 COMO COMPILAR:

1️⃣ CONFIGURE OS TOKENS (UMA VEZ):

   python configurar_tokens.py

   Informe:
   - TELEGRAM_BOT_TOKEN
   - TELEGRAM_CHAT_ID


2️⃣ COMPILE:

   .\build_standalone.bat

   OU manualmente:

   pyinstaller --noconfirm --onefile --windowed --name "Shopee-Telegram-BOT" --add-data "app;app" --hidden-import=tkinter --hidden-import=telegram --hidden-import=telegram.ext --hidden-import=requests --collect-all telegram main_app.py


3️⃣ PEGUE O .EXE:

   dist\Shopee-Telegram-BOT.exe  ← ESTE É O ARQUIVO FINAL!


4️⃣ DISTRIBUA:

   ✅ Copie APENAS o .exe
   ✅ Envie para quem quiser
   ✅ Não precisa enviar mais NADA junto!


📌 ESTRUTURA CRIADA AUTOMATICAMENTE:

Quando alguém executar o .exe, ele criará:

[Pasta do .exe]/
├── Shopee-Telegram-BOT.exe  ← A pessoa coloca aqui
├── downloads/                ← Criado automaticamente
├── processed/                ← Criado automaticamente
└── data/                     ← Criado automaticamente
    ├── videos_original.db
    └── videos_processados.db
